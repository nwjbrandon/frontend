## Install
- Install nvm
```
brew install nvm
```
- Install node v10.16.0
```
nvm install 10.16
nvm use 10.16
```
- Install yarn
```
brew install yarn
```

## Run
- Install dependencies
```
yarn install
```
- Start frontend
```
yarn start
```

## Upgrade
- Upgrade dependencies
```
yarn upgrade-interactive --latest
```
